## 3. Strategie GEO : comment les LLMs fonctionnent

### Comment un LLM decide quoi repondre
Les LLMs (ChatGPT, Gemini, Claude, Perplexity) construisent leurs reponses a partir de :

1. **Knowledge graphs** (Wikidata, Wikipedia) : source primaire pour les entites (personnes, oeuvres, lieux)
2. **Texte web indexe** : pages avec du contenu riche, structure, factuel
3. **Citations croisees** : quand plusieurs sources independantes mentionnent la meme personne
4. **Structured data** (Schema.org) : aide les crawlers a comprendre qui tu es
5. **Recence et frequence** : contenu recent = plus de poids
6. **llms.txt** : fichier dedicate qui guide les crawlers IA

### Perplexity specifiquement
Perplexity cherche en temps reel sur le web. Pour y apparaitre :
- Contenu indexable et recent sur ton site
- Mentions dans des sources tierces (presse, Wikipedia, bases de donnees)
- Structured data clair

### Google AI Overviews
Google AI Overviews puisent dans :
- Les resultats Google classiques (SEO traditionnel)
- Knowledge Panel (Google Knowledge Graph, alimente par Wikidata)
- Contenu structure et citable
